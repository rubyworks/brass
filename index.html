A Bare-Metal Ruby Assertion System Standard

BRASS is a very basic assertions framework for the Ruby programming language which provides a standardized system for assertion and test frameworks to utilize so that all such systems can work together harmoniously.

BRASS has been well thought-out and has been used in practice as part of other test framworks for years. Click here to learn more...
